XML 转 JSON

输入 XML，自动转换为 JSON 格式

XML 转 JSON 介绍

BeJSON XML 转 JSON是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 SOAP/WebService 返回的 XML 响应转为 JSON 用于现代前端框架
  • Java Maven 项目中 pom.xml 或配置文件的 XML 数据提取和分析
  • RSS/Atom 订阅源解析为 JSON 格式用于聚合阅读器开发
  • 遗留系统集成时将 XML 配置迁移为 JSON 格式

如何使用

  1. 在输入框粘贴 XML 数据
  2. 点击"转换"按钮解析 XML 结构
  3. 复制生成的 JSON 结果到项目中使用

常见问题

XML 属性会保留吗？

XML 元素的属性会以 @ 前缀形式保留在生成的 JSON 对象中。

同名子元素如何处理？

同名子元素会自动转为数组格式，确保数据不丢失。

支持 CDATA 段吗？

CDATA 内容会被提取并作为纯文本值处理。
