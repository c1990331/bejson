JSON 转 Properties

输入 JSON，转换为 Java Properties 格式

JSON 转 Properties 介绍

BeJSON JSON 转 Properties是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON 转为 Java Spring Boot 的 application.properties 格式
  • Java 微服务中将扁平化的键值对转为 Properties 文件
  • 国际化(i18n)资源文件 (.properties) 的快速生成和维护
  • 嵌套 JSON 自动展平为 dot 分隔的属性键名

如何使用

  1. 在输入框粘贴 JSON 数据（建议为单层对象）
  2. 点击"转换"按钮
  3. 复制生成的 Properties 内容到 .properties 文件中

常见问题

嵌套对象如何处理？

嵌套对象会被展平为点分隔的键路径，如 user.address.city → user.address.city

支持中文值吗？

支持。中文值会被原样保留，无需额外转义。

Properties 文件编码要求？

建议使用 ISO-8859-1 或 UTF-8 编码保存，以正确支持非 ASCII 字符。
