JSON 校验工具

检查 JSON 语法错误，提供详细错误信息

JSON 校验工具 介绍

BeJSON JSON 校验工具是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 校验后端 API 返回的 JSON 数据是否符合预期格式
  • 检查用户通过前端表单提交的 JSON 是否合法
  • 排查 JSON 解析错误原因，定位缺少引号或逗号的位置
  • 验证配置文件（如 tsconfig、eslintrc）的 JSON 语法正确性

如何使用

  1. 在输入框中粘贴需要校验的 JSON 数据
  2. 点击"校验"按钮进行语法检查
  3. 查看校验结果，如有错误会提示具体位置和原因

常见问题

JSON 校验和 JSON Schema 校验有什么区别？

JSON 校验只检查语法正确性（如括号匹配、逗号问题），而 JSON Schema 校验可以进一步验证数据类型、必填字段、值范围等业务规则。

支持带注释的 JSON 吗？

标准 JSON 不支持注释。如果您的 JSON 包含 // 或 /* */ 注释，建议先移除注释后再校验。

校验失败时如何快速定位错误？

工具会在结果区域显示具体的错误信息和位置，您也可以配合使用 JSON 格式化工具先美化再逐段排查。
