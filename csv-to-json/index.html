CSV 转 JSON

输入 CSV，自动转换为 JSON 格式

CSV 转 JSON 介绍

BeJSON CSV 转 JSON是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 Excel 导出的 CSV 数据转为 JSON 格式用于前端图表库（如 ECharts、Chart.js）
  • 数据库导出的 CSV 备份文件转换为 JSON 用于 API 接口返回
  • 电子表格数据与 Web 应用之间的数据交换和格式桥接
  • 批量导入 CSV 产品目录到电商系统或 CMS 后台

如何使用

  1. 在输入框粘贴 CSV 数据（首行为表头）
  2. 点击"转换"按钮自动识别表头和字段
  3. 复制生成的 JSON 数组到项目中使用

常见问题

CSV 表头必须是第一行吗？

是的。工具会将第一行作为 JSON 对象的键名，后续行作为值生成对象数组。

支持带引号的 CSV 吗？

支持标准 RFC 4180 CSV 格式，包括引号包裹的字段值。

空单元格如何处理？

空单元格会生成空字符串值 "" 而非 null。
