JSON 转 TOML

输入 JSON，自动转换为 TOML 格式

JSON 转 TOML 介绍

BeJSON JSON 转 TOML是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON 配置文件转换为 Rust/Cargo 项目使用的 TOML 格式
  • Hugo / Zola 等静态站点生成器的配置格式转换
  • Python 项目中 pip 工具的 pyproject.toml 配置生成
  • 将嵌套 JSON 扁平化为 TOML 的点分键路径结构

如何使用

  1. 在输入框粘贴 JSON 数据
  2. 点击"转换"按钮
  3. 复制生成的 TOML 内容到配置文件中使用

常见问题

TOML 和 JSON 有什么区别？

TOML 更注重可读性和注释支持，适合配置文件；JSON 更适合数据交换和 API 通信。

数组如何表示？

JSON 数组会转为 TOML 的内联数组格式（如 items = [1, 2, 3]）。

支持嵌套对象吗？

支持。嵌套对象会使用 [section.name] 方括号语法表示层级。
