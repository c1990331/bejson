JSON 转 Java

输入 JSON，自动生成 Java 实体类

JSON 转 Java 介绍

BeJSON JSON 转 Java是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 API 返回的 JSON 数据快速转为 Java POJO 实体类
  • Spring Boot 项目中对接第三方接口时的数据模型生成
  • Android 开发中解析服务端 JSON 响应构建数据类
  • 支持 Lombok 注解简化 Getter/Setter 样板代码

如何使用

  1. 在输入框粘贴 JSON 数据
  2. 设置目标 Java 类名，可选启用 Lombok 注解
  3. 点击"转换"按钮获取生成的 Java 代码并复制

常见问题

Lombok 注解有什么用？

开启后生成的类会添加 @Data 注解，自动生成 getter/setter/toString/equals/hashCode 方法，大幅减少样板代码。

支持嵌套对象吗？

支持。嵌套的 JSON 对象会自动拆分为独立的内部静态类。

数组类型如何映射？

JSON 数组会映射为 List<T> 类型，T 由数组元素类型决定。
