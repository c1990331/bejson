JSON 转 JavaScript

输入 JSON，生成 JavaScript 对象声明

JSON 转 JavaScript 介绍

BeJSON JSON 转 JavaScript是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON 配置数据转为 JavaScript const 变量用于前端项目
  • Node.js 脚本中将 JSON 文件内容转为 JS 模块引用
  • HTML 页面内嵌数据初始化时将 JSON 转为 JS 对象字面量
  • Mock 数据生成时快速创建 JS 测试数据常量

如何使用

  1. 在输入框粘贴 JSON 数据
  2. 点击"转换"按钮
  3. 复制生成的 JavaScript 代码直接使用

常见问题

输出的是合法 JS 语法吗？

是的。输出的代码使用 const 声明和 JSON.stringify 格式化，可以直接在浏览器或 Node.js 中运行。

和 JSON.parse 有什么区别？

JSON.parse 是运行时解析字符串；本工具是在编码阶段将 JSON 转为可读性更好的 JS 对象字面量源码。

支持 ES Module 导出吗？

当前输出 const 声明。您可以在生成的代码前加上 export 即可作为 ES Module 使用。
