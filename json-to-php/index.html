JSON 转 PHP

输入 JSON，自动生成 PHP 数组

JSON 转 PHP 介绍

BeJSON JSON 转 PHP是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON API 响应转为 PHP 关联数组用于 Laravel/ThinkPHP 项目
  • WordPress 插件开发中处理 JSON 配置数据
  • PHP 后端接收前端 JSON 请求体后的数据结构化
  • 生成可直接 json_decode 兼容的 PHP 数组语法

如何使用

  1. 在输入框粘贴 JSON 数据
  2. 点击"转换"按钮
  3. 复制生成的 PHP 数组代码到项目中使用

常见问题

输出的是索引数组还是关联数组？

JSON 对象转为关联数组（=>），JSON 数组转为索引数组，保持原有数据结构不变。

字符串中的特殊字符会被转义吗？

是的，单引号等特殊字符会自动转义，确保生成的 PHP 代码语法正确。

可以直接代替 json_decode 吗？

建议运行时仍使用 json_decode 解析动态数据。此工具主要用于生成静态配置或测试用的 PHP 数组字面量。
