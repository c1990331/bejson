什么是 JSON？

JSON（JavaScript Object Notation）是一种轻量级的数据交换格式，易于人类阅读和编写，同时也易于机器解析和生成。它基于 JavaScript 编程语言的一个子集，但独立于语言，几乎所有主流编程语言都提供了解析和生成 JSON 数据的能力。

对于前端开发者、后端工程师、测试人员以及数据分析师来说，日常工作中经常需要格式化杂乱的 JSON 数据、验证 API 返回的 JSON 结构是否正确、或者将 JSON 转换为 CSV/XML/SQL 等格式。BeJSON 提供了一站式解决方案，让您无需下载安装任何软件，打开浏览器即可使用。

不会。BeJSON 是纯前端应用，所有数据处理都在您的浏览器中完成，不会上传到服务器，刷新页面后数据即清除。
