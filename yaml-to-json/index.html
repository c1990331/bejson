YAML 转 JSON

输入 YAML，自动转换为 JSON 格式

YAML 转 JSON 介绍

BeJSON YAML 转 JSON是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 Docker Compose / Kubernetes / GitHub Actions 的 YAML 配置转为 JSON 用于程序读取
  • CI/CD 流水线中将 YAML 配置参数转为 JSON 传递给下游任务
  • Ruby on Rails / CircleCI 等项目的 YAML 配置解析和数据提取
  • Ansible Playbook 的 YAML 清单转为结构化的 JSON 数据

如何使用

  1. 在输入框粘贴 YAML 数据
  2. 点击"转换"按钮解析 YAML 层级结构
  3. 复制生成的 JSON 结果使用

常见问题

多文档 YAML 支持吗？

当前支持单个 YAML 文档的转换。多文档（用 --- 分隔）建议拆分后逐个转换。

锚点引用怎么处理？

YAML 锚点（&ref）会被解析为字符串值保留。

布尔值兼容吗？

true/false/null/yes/no 等常见布尔值都会正确映射为 JSON 布尔类型。
