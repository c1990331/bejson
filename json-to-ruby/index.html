JSON 转 Ruby

输入 JSON，生成 Ruby Hash

JSON 转 Ruby 介绍

BeJSON JSON 转 Ruby是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON API 数据转为 Ruby Hash 用于 Rails/Sinatra 后端开发
  • Rails 项目中解析前端提交的 JSON 请求数据
  • Jekyll 静态站点中将 JSON 配置转为 Ruby 数据结构
  • Ruby 脚本处理 JSON 数据时生成可读的 Hash 字面量

如何使用

  1. 在输入框粘贴 JSON 数据
  2. 设置变量名称（默认 data）
  3. 点击"转换"获取 Ruby Hash 代码并复制

常见问题

输出的符号是什么？

使用 :symbol => value 的 Ruby Hash 语法（Rocket 符号），这是 Ruby 中定义 Hash 的标准写法。

可以直接在 Rails 使用吗？

可以。生成的代码是纯 Ruby 语法，可直接复制到控制器或模型中使用。

Symbol 键会被转义吗？

字符串值中的引号会被正确转义为 \"，确保生成的 Ruby 语法合法。
