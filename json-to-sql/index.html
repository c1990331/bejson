JSON 转 SQL

输入 JSON，自动生成 INSERT 语句

JSON 转 SQL 介绍

BeJSON JSON 转 SQL是一款免费在线工具，无需注册，浏览器端直接使用，数据不会上传到服务器，保障您的隐私安全。

使用场景

  • 将 JSON 测试数据/Mock 数据快速生成 SQL INSERT 语句用于数据库初始化
  • Excel/CSV 导出的数据经 JSON 中转后批量导入数据库表
  • API 返回的 JSON 数组批量写入 MySQL/PostgreSQL/SQLite
  • 开发测试环境快速构建种子数据（Seed Data）

如何使用

  1. 在输入框粘贴 JSON 数组（每个元素为一行记录）
  2. 设置目标数据库表名
  3. 点击"转换"获取 INSERT INTO 语句并执行到数据库

常见问题

支持哪些数据库方言？

当前生成标准 SQL INSERT 语句，兼容 MySQL、PostgreSQL、SQLite、SQL Server 等主流数据库。

字符串中的引号会被转义吗？

是的，字符串值中的单引号会被自动转义为 \\'，确保 SQL 语句合法。

可以生成 UPDATE 或 DELETE 吗？

当前仅支持 INSERT 生成。UPDATE/DELETE 建议手动编写或使用 ORM 框架。
